public class State

Member Data

public List<string> CurrentKBs

- List containing the filepaths of all current KnowledgeBases.[Verification needed]

public Hashtable Vars

- Hash table containing all variables used throughout runtime. - Initialized with (and will always contain) the following:
Key Value
"_input" The input given this iteration.
"_lastinput" The last input given.
"_lastfired" The rule ID of the last rule fired.
"_time" DateTime.Now.ToString("h:mm tt");
"_time24" DateTime.Now.ToString("HH:mm");
"_date" DateTime.Now.ToString("MMM. d, yyyy");
"_month" DateTime.Now.ToString("MMMM");
"_dayofmonth" DateTime.Now.ToString("d ").Trim();
"_year" DateTime.Now.ToString("yyyy");
"_dayofweek" DateTime.Now.ToString("dddd");

- Note: to use the variables in Vars in the <?csharp ?> block, use the variable name vars instead. This is because the command parser CSharpToolbox uses vars (as the function parameter name).

public string Lastfired

- String containing the last fired rule's Rule ID.

public string Lastinput

- String containing the last input.

public DateTime LastRefreshedTime

- DateTime containing the timestamp of the last refresh.
- Always set to DateTime.Now on launch.
- This only applies to Verbots Online.


Member Functions

public void LoadVars(string filepath)

- Tries to load variables from filepath filepath. The file given in filepath should be one serialized using BinaryFormatter.

public void SaveVars(string filepath)

- Tries to save variables to filepath filepath. Serializes this.Vars using BinaryFormatter.
