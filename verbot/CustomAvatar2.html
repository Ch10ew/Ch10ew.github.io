Setting Your Custom Avatar (B)

What is a .ccs / .vsk file?

.ccs and .vsk files are just XML files in disguise. Hence, we are going to completely avoid using the Conversive Character Studio program in favour of writing our own files.

In this guide, we will be using Notepad++. However, this can be done in notepad as well.

The tags to note in this file are:

<CharacterFile> - filepath of the .ccs file. Can be absolute or relative, better for it to be relative like in the example.
<CharacterTTSMode> - The voice to use. Refer to voices.

The other tags are self explanatory by their names.


Voices

Voices used are Windows Text to speech defaults. There are open source voices but that's another topic. Most people would have these as their default:

  • 0 - Microsoft David Desktop - English (United States)
  • 1 - Microsoft Zira Desktop - English (United States)

You can get more TTS voices by installing random languages on your system. Some languages come with TTS voices. For example, Japanese comes with Microsoft Haruka Desktop - Japanese, in which case this will be at index 2.


The <CharacterTTSMode> tag uses the index of the TTS voice, as in if it is set to 0, Microsoft David Desktop will most likely be used as it is most people's default TTS voice (Refer to above list).

In the case where a number exceeding the amount of TTS voice available is given; e.g. 2, when only index 0 and 1 are available; the largest valid index will be used, in this case index 1.
