Setting Your Custom Avatar

by Rectorez / Kar Hou

As some may have noticed, Verbot5 in its current state doesn't support additional custom-made avatars due to .NET Framework issues.

While there is no known way of resolving this issue while retaining the current .NET Framework version, we can manipulate the existing files in the "Conversive Characters" folder. Specifically, the "julia" character, as she is the default file loaded by the Verbot 5 Player.
Folder directory showing the "julia" folder in the "Verbots/Conversive Characters" directory.

Once the "julia" folder has been opened, there should be the "images" folder. Open it and you'll find all of julia's images. You'll notice that these are the images you can trigger as julia speaks, such as speech and emotional expressions. Another neat thing to note is the fact that some of the pictures are not completely drawn.
The "images" folder shows all sprites used for the "julia" avatar.
